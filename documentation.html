Common Elements List

  • [] Html

    • The html element is requried in every html page.

    (!DOCTYPE html)

    This tells the server to use html 5.

    html lang="en"
    /html

    This tells the server to use english. All code should be between the html tags.

  • [] Head
    • head
    /head

    The head tags is where the websites title, style, and meta are.
  • [] Style
    • style
    /style

    This is where the css code for all the elements can be put creating a uniquie page design.

    Common css code inclues

    • background-color:blue;

      • Creates the background color of the element.

    • color:white;

      • Assigns the color of the text within the element.

    • border-radius:12px;

      • This is how to make the elements have rounded borders

    • text-align:center;

      • This aligns the texted within the element. It can be right, left or center.
  • [] Title

    • Title is found with in the head element and creates the title of the webpage.

  • [] Meta

    • Meta is used to create a responsive page that can be viewed on all devices.

  • [] Body

    • After the head element you have the body element. The body element makes up the parts of the page that you can see while visiting the site.

  • [] Main

    • The Main element wraps around the main content of the page.

  • [] Div

    • Div is an element that holds a section of the webpage and helps to divied content allowing for an organized layout.

  • [] Lists

    • Lists can be ordered or unordered.

    ul
    /ul

    The ul tag creates an unordered list and would have list items or

    li
    /li

    tags that hold the content of the list.

    ol
    /ol

    The ol tags created an ordered list which is marked by numbers instead of dots like and unordered list.
  • [] Headings

    • Headings can be created using

    h1 - h6

    each number is a different size but also helps screen readers tell the difference between main heading and subheadings.
  • [] Anchors

    • Achors link an oject to either another part of the page or an external link.

    a href="link goes here"
    /a
  • [] Image

    • This element helps to add an image to the page

    img src="link here" alt="alternative view"
    /img

    For an image you need the src that is either stored on the same webstie or a link to an external site. It is also considered madatory to have a alt sentence describing the picture for users who are using screen readers.


Survey Page

New Elements Introducted

The survey.html page introduces some new elements they are as follows.

  • [] Form

    • The form element creates a section of the page that can be used to get input from the user. Such as name, email, age, phone number. Really what ever information you would like.

    form
    /form

    All of the code for the form including input, label, select, option, and button should all go between the form tags.
  • [] Label

    • Label is used to create your guessed it a label. The label can provide a visible name for another element like and input element within the form

    label for=""
    /label
  • [] Input

    • Input is the element that will actullay determine what input you get from the user a few common examples inlcude:

    input type="text"

    Examples of input types

    • text
    • radio
    • checkbox
    • submit
    • email
    • password
    • numbers
    input name="ex"
    /input

    Name allows the input to be recored under that specific value. A list of all users emails might be name="email" with type="email"

    input placeholder="name"

    placeholders are used with in the form to tell the user what type of data is requested such as email or name.

  • [] Select

    • The select element creates a dropdown list that users can choose from. Within the select element there need to be option elements.

    select label option name="text" value="text" /label /select
  • [] Option

    • As you can see in the code above the option tag is used to create options with in the select tag for a drop down list.

  • [] Button

    • The button element creates you guessed it a button

    I also use anchor tags with in this tage to direct the user to a new page.

    button id="name" a href="link" /a /button
Landing Page

New Elements Presented in Landing.html

The following are a few common elements that were presented in the landing.html page

  • [] Nav
    • nav id="#" div class="nav-item" button a href="#" text /a /button /div /nav

    As you can see above the nav element can create a navigation bar and within it has div with buttons and links

  • [] Video
    • video width="100" height="300" controls source src="link" type=".." /video

    The video element defines the height and width of the viewer to see the video, controls shows the controls for the video and then source of course provides the file path or length as well as the type of video file.

  • [] Iframe
    • iframe width="" hieght="" src="" /iframe

    I frame is very similar to the video element but is used specifically for youtube videos. As seen here you still define the hieight and width as well as the link so that useres can view the video while still on your page.

Fun Extras

You have already seen that I love to use rounded edges on all by elements. I just like the look better. This is completed by:

You can also make your background an image instead of a color by doing this:

body { background-img: url('link goes here');

Also helpful tip that will make your html look less cluttered. You do not need to put all your style code in the head section. You can instead create a stylesheet.css and keep all you style code in that file. This will allow you to decrease clutter on you html page and also to use the css code on multiple pages. All you need to do is reference the link in the head element as follows

link rel="stylesheet" href="stylesheet.css"
